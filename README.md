# Microsoft Entra ID IAM Lab
## Project Overview
This hands-on Microsoft Entra ID Identity and Access Management (IAM) lab was completed to develop practical experience with identity administraion, user lifecycle management, security groups, role-based access control (RBAC), access reviews, audit logging, user provisioning, and user deprovisioning.

## Skills Demonstrated
- Microsoft Entra ID Administration
- Identity & Access Management (IAM)
- User Lifecycle Management
- Role-Based Access Control (RBAC)
- Access Reviews
- Authentication Management
- Password Management
- Audit Log Analysis
- User Provisioning
- User Deprovisioning
- Documentation and Change Tracking

## Supporting Evidence
The repository contains:
- Project Overview Documentaion
- Tenant Configuration Review
- RBAC Configuration and Validation
- Access Review Activities
- Identity Auditing Activities
- User Deprovisioning Activities

## Technologies Used
- Microft Entra ID
- Azure Portal
- Security Groups
- RBAC
- Audit Logs
- Sign-in Logs
- Access Reviews

## Purpose
The objective of this project was to gain practical experience with identity administration processes commonly performed by IAM analysts and administrators in enterprise environments.
